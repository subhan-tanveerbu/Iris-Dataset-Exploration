# Iris Dataset Exploration

## Objective
To analyze and visualize the Iris dataset to understand patterns and feature relationships.

## Dataset
The Iris dataset contains 150 samples of iris flowers with 4 numerical features and 1 categorical target variable.

## Tasks Performed
- Data inspection using pandas
- Summary statistics
- Scatter plot visualization
- Histograms with different colors
- Boxplot for outlier detection
- Pairplot for feature comparison

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Conclusion
Petal length and petal width are strong features for distinguishing iris species.
